Über MySQL Backup Manager

Der MySQL Backup Manager ist ein Tool zur einfachen Sicherung von MySQL-Datenbanken. Es bietet folgende Funktionen:

  • Konfiguration der MySQL-Datenbankverbindung
  • Backup auf lokalen Speicher oder SMB-Netzwerkfreigaben
  • Flexible Zeitplanung für automatische Backups
  • Einstellung der Aufbewahrungsdauer für Backups
  • Manuelle Ausführung von Backups bei Bedarf
Erste Schritte

Um mit dem MySQL Backup Manager zu beginnen, folgen Sie diesen Schritten:

  1. Gehen Sie zur Konfiguration und richten Sie die MySQL-Verbindung ein.
  2. Konfigurieren Sie optional einen SMB-Share als Backup-Ziel.
  3. Legen Sie im Zeitplan fest, wann automatische Backups ausgeführt werden sollen.
  4. Führen Sie ein erstes manuelles Backup durch, um die Konfiguration zu testen.
  5. Überprüfen Sie die erstellten Backups auf der Backups-Seite.
